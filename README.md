## 🌟 О проекте / Project Overview

**librarybookingbot** — это инструмент для цифровизации работы библиотеки IRC O'zDJTU. Он упрощает процесс взаимодействия студентов с ресурсами центра.

### 📝 Описание контента (Content Description):
Данный репозиторий содержит исходный код Telegram-бота, который позволяет:
* Просматривать доступность учебных залов в режиме реального времени.
* Резервировать места для групповых и индивидуальных занятий.
* Получать доступ к базе данных учебных материалов центра.

### ☕ Поддержка и развитие (What the audience supports):
Ваша поддержка через **Buy Me a Coffee** помогает мне:
* Оплачивать хостинг и сервера для бесперебойной работы бота.
* Разрабатывать новые модули (интеграция с электронными картами читателей).
* Поддерживать мои технические исследования в области автоматизации образовательных процессов.
